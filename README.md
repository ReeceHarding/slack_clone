# Modern Chat Application

A real-time chat application built with Next.js, Convex, and modern web technologies.

## Features
- Real-time messaging
- Workspace management
- Channel & direct messaging
- File sharing
- User authentication
- Modern UI with shadcn/ui

## Tech Stack
- Next.js
- Convex
- TypeScript
- shadcn/ui
- TanStack Query
- Tailwind CSS
